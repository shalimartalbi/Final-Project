OneSignal Workspace Prototype
Overview

This project is an interactive front-end prototype for OneSignal, simulating a unified workspace that integrates email and calendar management into a single interface.

The prototype demonstrates UI interactions and workflow simulation using HTML, CSS, and JavaScript.

Key Features

Email

Open, reply, forward, and send emails

Conversation threads with editable composer

Filters: All, Unread, Important, Sent, Archive

Flag, archive, and mark emails as read/unread

Simulated attachment downloads

Calendar

View scheduled meetings

Create and cancel events

Join meetings

Event detail popups (participants, location, notes)

Custom date filtering

Overview Dashboard

Displays unread emails and upcoming meetings

Quick navigation to email or calendar

Counts update dynamically based on interactions

Other Interactions

Connect tools (Email or Calendar)

Simulated login/logout flow

Basic settings screen
